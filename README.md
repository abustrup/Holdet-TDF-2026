# Holdet TDF 2026 — bot dashboard

Live interpretability dashboard for the autonomous holdet.dk Tour de France 2026 fantasy bot
("The Times They Are A Changin'"). It shows the current team, why each rider was chosen, the
strategy, the EV substrate, and the bot's known weaknesses.

**View:** https://abustrup.github.io/Holdet-TDF-2026/

`index.html` is a self-contained page (data inlined, no external requests), rebuilt from the bot's
live data and refreshed daily. Read-only.
